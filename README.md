# Signature Collection- متجر الإلكتروني

### فكرة المشروع :

موقع ويب لمتجر إلكتروني يعرض أحدث الأجهزة الذكية والالكترونيات، ويحتوي على ميزة التصفية الذكية (Filter)
للتنقل بين الأقسام المختلفة بسلاسة .

### الأدوات المستخدمة :

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
- FontAwesome (الأيقونات)
